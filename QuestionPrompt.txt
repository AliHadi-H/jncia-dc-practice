Tagged ( Trunk ) 

Used to allow multiple VLANS 

Usually links to another switch or an edge router 

It can handle traffic for multiple VLANs. Means that it can send and receive data for all the configured VLAN's over the same physical connection. 

Sends the traffic for all configured VLANs over the same physical connection and separates traffic by tagging it with appropriate VLAN ID 

Can carry untagged traffic when configured with the native-vlan-id statement.
